# Ship - Open Collective Application

## About Ship

**Ship** is an open-source autonomous AI coding agent that revolutionizes how developers work with GitHub. By assigning issues directly to Ship, developers can watch as it autonomously writes code, creates pull requests, and responds to feedback â€” all in the background.

### Our Mission

To democratize AI-powered development tools and make every developer 10x more productive by automating repetitive coding tasks while maintaining code quality and security.

---

## The Problem We Solve

Developers spend significant time on:
- Writing boilerplate code
- Creating and managing pull requests
- Responding to code review feedback
- Fixing repetitive bugs
- Managing issue backlogs

**Ship automates these tasks**, allowing developers to focus on creative problem-solving and innovation.

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Autonomous Coding** | AI writes code based on issue requirements |
| **Auto PR Creation** | Creates well-formatted pull requests |
| **Feedback Response** | Responds to code review comments |
| **Security Scanning** | Find and fix vulnerabilities in real time |
| **Multi-LLM Support** | Works with OpenAI, Claude, Groq, Ollama |
| **One-Click Deploy** | Transform ideas to deployed apps |

---

## Why Open Source?

We believe AI coding tools should be:

1. **Accessible** - Available to all developers, not just enterprises
2. **Transparent** - Open code means trustworthy AI
3. **Community-Driven** - Built by developers, for developers
4. **Customizable** - Adaptable to different workflows
