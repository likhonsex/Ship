## What is Ship?

**Ship** is an autonomous AI coding agent that works directly in your GitHub workflow. Assign issues to Ship and watch it:

- Write code based on requirements
- Create pull requests automatically
- Respond to code review feedback
- Deploy with a single click

## Quick Start

```bash
git clone https://github.com/likhonsex/Ship.git
cd Ship
npm install
npm run dev
```
